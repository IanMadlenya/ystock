yStock

This gem provides you with the latest market data available, pulling from both Google and Yahoo Finance.

Install

Add the following to your Gemfile

gem "ystock", "~> 0.4.5"

# Or simply install it
gem install ystock -v=0.4.5

Bundle

bundle install

Sinatra or Ruby App

require 'ystock'

Yahoo Usage

Quote Usage

# Single Stock lookup {String}
Ystock::Yahoo.quote("appl")

# Multiple Stock lookup {Array}
Ystock::Yahoo.quote(["aapl", "f", "goog"])
